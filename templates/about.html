About This Project

This web application predicts garment worker productivity based on operational inputs using a machine learning model trained on real industry data.

  • Incentive: Monetary bonus given to worker (0–infinity)
  • Over Time: Extra time worked beyond regular hours
  • Number of Style Changes: Times style changed in a day
  • SMV: Standard Minute Value – time required to complete a task
  • Month: Numeric (1 for January, 12 for December)
  • Weekday: 0 = Monday to 6 = Sunday
  • Department: 1 for Sewing, 0 for Finishing

Output: Predicted productivity score (between 0 and 1).
